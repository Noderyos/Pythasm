Video who explained the full project step by step :  https://www.youtube.com/watch?v=8g3RBZFb1Tc

## Build

To build the project you need 
- make
- nasm
- gcc
- ld

Under debian : `sudo apt install nasm gcc binutils make`

Just run `make`
